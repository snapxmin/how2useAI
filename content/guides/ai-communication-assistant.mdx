## AI 能帮你解决什么沟通问题

很多职场沟通不是信息不够，而是表达不够清楚、语气不够合适、预期没有对齐。AI 可以辅助这些场景：

- 把口语化想法改成清晰表达
- 准备和领导、客户、跨部门同事的沟通话术
- 将强硬表达改得更专业
- 模拟对方可能追问的问题
- 生成会前沟通提纲和会后跟进消息

## 场景一：把想法说清楚

当你知道自己想表达什么，但写出来很乱时，可以用这个 Prompt。

```
请帮我把下面这段话改写得更清楚。

## 我的原话
[粘贴你的想法]

## 沟通对象
[如：直属领导、客户、合作部门]

## 沟通目标
[如：说明进展、争取资源、推动对方确认]

## 要求
- 保留我的核心意思
- 逻辑顺序更清晰
- 语气专业但不生硬
- 输出 2 个版本：简洁版和正式版
```

## 场景二：准备关键沟通

在开会、汇报或谈需求前，可以先让 AI 帮你准备沟通提纲。

```
我要和 [沟通对象] 沟通 [主题]，请帮我准备沟通提纲。

## 背景
[说明当前情况]

## 我希望达成的结果
[说明你希望对方同意、确认或提供什么]

## 可能的分歧
[写出你担心的问题]

## 输出格式
1. 开场怎么说
2. 需要说明的 3 个重点
3. 对方可能追问的问题
4. 我的回应建议
5. 会后跟进消息模板
```

## 场景三：调整语气

需要催进度、提要求或拒绝不合理请求时，AI 可以帮你把表达变得更得体。

```
请帮我改写下面这段话，让语气更专业、坚定但不冒犯。

## 原文
[粘贴原文]

## 目标
[如：提醒对方今天给反馈 / 拒绝临时加需求 / 说明延期风险]

## 要求
- 不阴阳怪气
- 不过度道歉
- 明确下一步动作
- 控制在 120 字以内
```

## 使用边界

AI 可以帮你组织语言，但不能替你判断人际关系。以下情况要谨慎：

- 涉及绩效、投诉、法律或合同争议
- 对方情绪明显激烈
- 内容包含公司敏感信息
- 需要代表团队做承诺

## 推荐工具

- **豆包**：适合快速改写日常中文表达
- **Kimi**：适合长文本沟通场景和中文语境理解
- **ChatGPT**：适合生成多种语气版本
- **Claude**：适合复杂沟通场景和长上下文分析

沟通类任务最适合让 AI 先生成“可选说法”，再由你根据关系和场合做最后判断。
