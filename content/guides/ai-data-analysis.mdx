## 适合谁

- 经常处理 Excel 表格但不懂 Python 的职场人
- 需要快速从数据中提取洞察的产品/运营/市场人员
- 想做出专业图表但不会用 BI 工具的管理者

## 准备工作

1. 整理好你的数据文件（Excel 或 CSV 格式）
2. 明确你想回答的问题（如：「哪个产品线 Q2 增长最快？」）
3. 使用支持文件上传的 AI 工具：
   - **ChatGPT Plus**（Code Interpreter / Advanced Data Analysis）
   - **Kimi**（支持上传 Excel 文件）

## 实战案例：销售数据分析

假设你有一份 Q2 销售数据表，包含：日期、产品、区域、销售额、订单数。

### Prompt 1：数据概览

```
我上传了一份 Q2 销售数据。请帮我：
1. 概览数据基本情况（行数、字段、缺失值）
2. 计算总销售额和总订单数
3. 按产品分组，列出销售额 Top 5
4. 找出销售额最高的区域
```

### Prompt 2：趋势分析

```
基于刚才的数据：
1. 绘制月度销售额趋势折线图
2. 标注销售额最高和最低的月份
3. 计算月环比增长率
4. 用 2-3 句话总结趋势
```

### Prompt 3：异常检测

```
请检查数据中是否有异常：
1. 销售额为 0 或负数的记录
2. 单日销售额超过月均 3 倍的异常值
3. 某个区域数据突然断崖式下降
列出异常记录并给出可能原因
```

## 三个关键技巧

### 1. 分步提问，不要一次问太多

不要在一个 Prompt 里要求 10 个分析维度。先概览，再深入，再可视化，效果更好。

### 2. 要求 AI 展示计算过程

加上一句「请展示你的计算步骤和使用的公式」，方便你验证结果是否正确。

### 3. 导出结果

分析完成后，让 AI「将结果导出为 Excel 文件」或「生成可复制的 Markdown 表格」，方便粘贴到汇报中。

## 局限性

- AI 数据分析适合**探索性分析**和**快速洞察**，不适合需要严格审计的财务报告
- 数据量建议控制在 **10 万行以内**，过大的文件可能处理失败
- 复杂的多表关联分析仍建议学习基础的 SQL 或 Python
