## 什么是 Cursor

Cursor 是一款基于 VS Code 的 AI 原生代码编辑器。它不只是「代码补全」，而是能**理解你的整个项目**，帮你完成从需求到代码的全流程。

核心能力：
- **Tab 补全**：比 Copilot 更智能的多行代码补全
- **Ctrl+K 内联编辑**：选中代码，用自然语言描述修改
- **Agent 模式**：描述一个功能，AI 自动搜索代码库、编写、测试

## 安装与配置

### 1. 下载安装

访问 [cursor.com](https://cursor.com) 下载对应系统的安装包。Cursor 基于 VS Code，你的 VS Code 插件和设置可以一键导入。

### 2. 选择 AI 模型

设置 → Models，推荐配置：
- 日常编码：**Claude Sonnet**（平衡速度和质量）
- 复杂任务：**Claude Opus** 或 **GPT-4o**
- 快速补全：默认的 Cursor Tab 模型

### 3. 配置项目规则

在项目根目录创建 `.cursor/rules` 文件，告诉 AI 你的项目约定：

```
- 使用 TypeScript strict 模式
- 组件使用函数式写法
- 测试使用 Vitest
- 提交信息使用中文
```

## 三个核心操作

### 操作一：Tab 补全

正常写代码，Cursor 会预测你的下一行甚至下一个函数。按 `Tab` 接受建议。

**技巧**：先写一行注释描述你要做什么，Tab 补全的质量会大幅提升。

### 操作二：Ctrl+K 内联编辑

1. 选中要修改的代码
2. 按 `Ctrl+K`（Mac: `Cmd+K`）
3. 输入修改指令，如：「添加错误处理」「重构为 async/await」
4. 预览 diff，确认后 Accept

### 操作三：Agent 模式（重点）

1. 按 `Ctrl+I`（Mac: `Cmd+I`）打开 Agent 面板
2. 描述你的任务：

```
请为用户列表页添加搜索功能：
1. 在表格上方添加搜索输入框
2. 支持按姓名和邮箱模糊搜索
3. 搜索时实时过滤，不需要按回车
4. 添加对应的单元测试
```

3. Agent 会自动：搜索相关文件 → 阅读现有代码 → 编写新代码 → 运行测试
4. 你审核 diff，确认后应用

## 第一个实战任务

用 15 分钟完成这个练习：

1. 用 Cursor 打开任意一个现有项目（或新建一个）
2. 用 Agent 模式添加一个简单功能（如：API 健康检查端点）
3. 用 Ctrl+K 优化 Agent 生成的代码
4. 让 Agent 为新增代码编写测试

## 常见问题

**Q：Cursor 免费版够用吗？**
A：免费版每月有 2000 次补全和 50 次慢速 Agent 请求，个人学习完全够用。

**Q：Cursor 和 Copilot 怎么选？**
A：Copilot 适合「我写代码，AI 补全」；Cursor 适合「我描述需求，AI 写代码」。建议开发者两者都试，按场景切换。

**Q：Agent 模式会改坏我的代码吗？**
A：Agent 的所有修改都会显示 diff 预览，你可以逐文件 Accept 或 Reject。建议先用 Git 提交当前状态再使用 Agent。
