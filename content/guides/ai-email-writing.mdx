## 场景与收益

商务邮件是职场人最高频的写作任务之一。传统方式写一封得体的邮件通常需要 20-30 分钟（构思、起草、润色、检查）。

使用 AI 后，这个流程可以压缩到 **5 分钟**：

| 步骤 | 传统耗时 | AI 辅助耗时 |
|------|----------|-------------|
| 构思内容 | 10 分钟 | 1 分钟（口述要点） |
| 起草 | 10 分钟 | 1 分钟（AI 生成） |
| 润色检查 | 10 分钟 | 3 分钟（人工微调） |

这篇文章适合需要频繁写会议确认、项目沟通、客户回复和进度跟进的职场人。目标不是让 AI 替你判断，而是让 AI 承担起草和润色工作，把你的时间留给核对事实、确认立场和把控语气。

## 5 分钟写邮件工作流

### 第一步：先列清楚要点

不要一上来就让 AI “帮我写封邮件”。先用 1 分钟写清楚这些信息：

- **写给谁**：领导、客户、合作方、跨部门同事
- **为什么写**：确认、汇报、催进度、拒绝、说明风险
- **必须包含什么**：时间、事项、数据、下一步动作
- **语气是什么**：正式、友好、坚定、委婉
- **有什么边界**：不能承诺的内容、需要对方确认的事项

### 第二步：让 AI 生成可编辑初稿

好的 Prompt 要比“帮我写一封邮件”多给上下文。比如：

```
请帮我写一封商务邮件。

## 背景
[说明当前情况]

## 收件人
[如：客户项目负责人 / 直属领导 / 合作部门同事]

## 邮件目的
[如：确认会议时间 / 汇报项目进展 / 催促对方反馈]

## 必须包含的信息
- [要点 1]
- [要点 2]
- [要点 3]

## 语气
[如：专业友好、简洁直接、坚定但不冒犯]

## 要求
- 控制在 150-200 字
- 不要编造我没有提供的信息
- 如果有需要我补充的地方，用 [需确认] 标出
```

### 第三步：人工检查后再发送

AI 生成的是初稿，不是最终稿。发送前至少检查事实、承诺、语气和敏感信息，尤其是时间、金额、附件、客户名称和下一步动作。

## Prompt 好坏对比

普通写法：

```
帮我写一封会议确认邮件。
```

更好的写法：

```
请帮我写一封商务邮件，用于确认下周项目会议时间。

背景：我们正在推进 Q3 合作方案讨论。
收件人：合作方项目经理。
目的：确认会议时间，并提醒对方提前准备议题。
可选时间：周三或周四下午。
语气：专业、友好、简洁。
长度：150 字以内。
```

两者的差别在于：后者给了对象、目的、背景、限制和语气，AI 才能生成更接近真实工作场景的内容。

## 五种高频场景

### 场景一：从零起草

当你需要写一封全新的邮件时：

```
请帮我写一封商务邮件：

- 收件人：[对方职位/关系，如：合作方项目经理]
- 目的：[如：确认下周会议时间]
- 关键信息：[如：我提议周三或周四下午，讨论 Q3 合作方案]
- 语气：[如：专业友好]
- 语言：中文
- 长度：150 字以内
```

### 场景二：润色已有草稿

当你写了初稿但不够专业时：

```
请润色以下邮件草稿：
1. 保持我的核心意图不变
2. 语气更专业但不生硬
3. 修正语法和标点
4. 控制在 200 字以内

[粘贴你的草稿]
```

### 场景三：回复复杂邮件

当收到一封需要仔细回复的邮件时：

```
我收到了以下邮件，请帮我起草回复：

## 要求
- 对每条问题/请求逐一回应
- 语气：[如：积极合作但需明确边界]
- 如有不确定的事项，标注 [需确认] 让我补充

## 收到的邮件
[粘贴原文]
```

### 场景四：催进度但不显得强硬

当你需要提醒对方反馈、交付或确认事项时：

```
请帮我写一封催进度邮件。

## 背景
[说明之前约定的事项和时间]

## 目标
提醒对方尽快反馈，并明确下一步安排。

## 要求
- 语气专业、坚定但不冒犯
- 不过度道歉
- 明确希望对方在什么时间前完成什么动作
- 控制在 150 字以内

## 关键信息
[粘贴事项、截止时间、影响范围]
```

### 场景五：会后跟进和行动项确认

会议结束后，可以让 AI 帮你整理一封跟进邮件：

```
请帮我写一封会后跟进邮件。

## 会议主题
[填写会议主题]

## 需要确认的结论
- [结论 1]
- [结论 2]

## 待办事项
- [负责人]：[事项]，[截止时间]
- [负责人]：[事项]，[截止时间]

## 要求
- 结构清晰，方便对方逐条确认
- 语气专业、简洁
- 对不确定的信息标注 [需确认]
```

## 发送前检查清单

发送 AI 生成的邮件前，至少检查 5 件事：

- [ ] 有没有 AI 编造的事实、数据、承诺或截止时间？
- [ ] 日期、时间、地点、金额、附件是否准确？
- [ ] 是否逐一回应了对方真正关心的问题？
- [ ] 语气是否符合你和对方的关系？
- [ ] 是否包含客户姓名、合同金额、内部决策等敏感信息？

## 注意事项

1. **永远人工终审**：AI 可能编造不存在的承诺或数据，发送前务必检查
2. **保护敏感信息**：不要在 Prompt 中包含客户真实姓名、金额等机密信息，用 [客户 A]、[XX 万] 替代
3. **保持个人风格**：用 AI 生成 2-3 个版本，选择最接近你说话风格的那个再微调

## 推荐工具

- **Kimi**：中文邮件首选，理解中文语境更准确
- **豆包**：适合快速起草日常中文邮件
- **ChatGPT**：英文邮件首选，表达地道
- **Notion AI**：如果邮件内容需要关联项目文档，在 Notion 中直接生成更方便

掌握这个流程后，AI 最适合帮你完成“初稿”和“改写”，但最终判断仍然要由你完成。更多 Prompt 模板请查看 [Prompt 库](/prompts) 中的「商务邮件润色」。
